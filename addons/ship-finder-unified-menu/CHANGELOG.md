# Changelog

## 1.0.0

- Added the unified `Ctrl+Alt+F` Ship Finder menu.
- Added automatic general/personal menu selection.
- Added current-province filtering for Latium and Albion.
- Added alphabetical, active-route, attention, runner, warship and other-ship categories.
- Added paging and direct camera jumps to selected ships.
- Added stable back navigation between categories.
- Removed the invalid generic Decision outcome panel.
