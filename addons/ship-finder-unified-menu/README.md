# Ship Finder - Unified Menu

A live ship-selection menu for **Anno 117: Pax Romana**.

Press **Ctrl+Alt+F** to open Ship Finder. Every menu is built from the current
province only, so Latium shows Latium ships and Albion shows Albion ships.

## General menu

- Alphabetical Ship List
- Active Trade Routes
- Trade Ships Needing Attention
- Warships
- Other Ships
- Close

## Personal naming menu

When at least one current-corporation ship follows either of these patterns:

- `L_<first island>_<second island>_<number>`
- `A_<first island>_<second island>_<number>`

Ship Finder automatically enables the personal menu:

- Alphabetical Ship List
- Active Trade Routes
- Trade Ships Needing Attention
- Runners
- Warships
- Other Ships
- Close

Active Trade Routes means assigned and not paused. Trade Ships Needing
Attention means paused or unassigned.

## Installation

This add-on requires the core mod with ModID `ship-finder-sirlocksley`.

Install both the core Ship Finder mod and this unified-menu add-on in the active
Anno 117 mods folder, then restart the game.

## Shortcut

- `Ctrl+Alt+F`: Open the unified Ship Finder menu

## Savegame safety

- Does not require a new game
- Safe to remove
- Queries live ships rather than storing savegame data
